üîÑ Startup Flow

  1. Boot: OLED shows "Starting..."
  2. Check Preferences: Look for stored WiFi credentials
  3. Check Run Local Flag: If "Run Locally" was pressed, go to AP mode
  4. Try WiFi: If credentials exist, attempt connection with OLED feedback
  5. Fallback to AP: If WiFi fails, start AP mode (no password)
  6. Start Web Server: All pages available in both modes
  7. Continuous Operation: OLED shows current network status
